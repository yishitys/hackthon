# Guidance Steps

## (0) DEFINE

One-page product brief before any code. Who's it for, what does it do (one sentence), what does success look like, what won't you build.

## (1) FIND IT

Agent 1 reads the data. Finds what's wrong.

## (2) RANK IT

Agent 2 sorts the findings. Worst first, with reasons.

## (3) ACT ON IT

Agent 3 fixes, flags, or escalates. Every action has a logged reason.

## (4) EXPLAIN IT

Agent 4 writes the summary a human can sign. Downloadable.

## (5) SHOW IT

Build the product from Step 0. Demo it as your user.

> "Memory connects everything - each agent recalls the previous one's work through Cognee."
